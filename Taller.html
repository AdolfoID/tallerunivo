📌 Fundamentos que debes dominar

Antes de programar, revisa estos conceptos. En clase los aplicarás construyendo soluciones reales con tu equipo.

🏷️ Clase

Una clase es una plantilla o molde que define atributos y comportamientos.

// Ejemplo de clase (JavaScript) class Mascota { constructor(nombre, especie) { this.nombre = nombre; this.especie = especie; this.energia = 100; } saludar() { return `¡Hola! soy ${this.nombre}`; } }

🎯 Objeto

Un objeto es una instancia concreta de una clase.

// Creando objetos (instancias) let perro = new Mascota("Max", "Perro"); let gato = new Mascota("Luna", "Gato"); console.log(perro.saludar()); // ¡Hola! soy Max

⚙️ Atributos y Métodos

Atributos: datos que describen al objeto.
Métodos: acciones que puede realizar.

// Atributos: nombre, especie, energia // Métodos: saludar(), jugar(), dormir()

🤝 Construyendo juntos: Sistema de Gestión

Cada equipo organizará sus roles para diseñar una clase funcional. La colaboración es clave.

📋 Roles del equipo (4 personas)

🎨 Rol 1

Analista de Atributos

Define qué datos necesita la clase (tipo de datos, valores iniciales, validaciones).

💻 Rol 2

Constructor de Métodos

Escribe las funciones y comportamientos (calcular, transformar, mostrar).

🧪 Rol 3

Probador / Depurador

Crea instancias, prueba los métodos y verifica que funcionen correctamente.

🎤 Rol 4

Relator / Presentador

Documenta el código y prepara la exposición del proyecto.

🎯 Misión del equipo: Clase "Producto"

Construyan una clase Producto con los siguientes requisitos:

  • Atributos: nombre, precio, stock, descuento (porcentaje)
  • Método precioFinal() que devuelva precio con descuento aplicado
  • Método vender(cantidad) que reduzca el stock y muestre mensaje
  • Método mostrarInfo() que retorne toda la información
// Clase Producto - Solución colaborativa class Producto { constructor(nombre, precio, stock, descuento = 0) { this.nombre = nombre; this.precio = precio; this.stock = stock; this.descuento = descuento; // porcentaje 0-100 } precioFinal() { let descuentoAplicado = this.precio * (this.descuento / 100); return this.precio - descuentoAplicado; } vender(cantidad) { if(cantidad <= this.stock) { this.stock -= cantidad; return `✅ Venta exitosa. Stock restante: ${this.stock}`; } else { return `❌ Stock insuficiente. Solo quedan ${this.stock} unidades.`; } } mostrarInfo() { return `📦 ${this.nombre} | Precio: S/ ${this.precio} | Oferta: ${this.descuento}% | Final: S/ ${this.precioFinal().toFixed(2)} | Stock: ${this.stock}`; } } // Prueba de concepto const laptop = new Producto("Laptop Gamer", 2500, 5, 15); console.log(laptop.mostrarInfo()); console.log(laptop.vender(2));

🚀 Proyecto: Sistema del Mundo Real con POO

Tu equipo elegirá uno de estos proyectos para construir una solución funcional usando clases y objetos.

📱 Opción 1: Gestor de Tareas

🎮 Opción 2: Personaje de Videojuego

Clase Personaje con vida, fuerza, nivel. Métodos: atacar(), subirNivel(), curar().

🏦 Opción 3: Cuenta Bancaria

Clase Cuenta con saldo, titular, número de cuenta. Métodos: depositar(), retirar(), mostrarSaldo().

📚 Opción 4: Biblioteca Digital

Clase Libro con título, autor, disponible. Métodos: prestar(), devolver(), info().

📋 Entregables del proyecto

  • ✅ Código de la clase con sus atributos y métodos (pseudocódigo o JavaScript)
  • ✅ Al menos 3 objetos creados con diferentes datos
  • ✅ Ejecución de métodos que demuestren funcionalidad
  • ✅ Breve presentación explicando decisiones de diseño
// Clase Personaje para videojuego class Personaje { constructor(nombre, vida, fuerza) { this.nombre = nombre; this.vida = vida; this.fuerza = fuerza; this.nivel = 1; } atacar(enemigo) { let daño = this.fuerza * this.nivel; enemigo.vida -= daño; return `${this.nombre} ataca causando ${daño} de daño. ${enemigo.nombre} tiene ${enemigo.vida} de vida restante.`; } subirNivel() { this.nivel++; this.fuerza += 5; return `${this.nombre} subió al nivel ${this.nivel}! Fuerza: ${this.fuerza}`; } curar() { this.vida += 20; return `${this.nombre} se cura. Vida actual: ${this.vida}`; } estado() { return `⚔️ ${this.nombre} | Nivel ${this.nivel} | ❤️ Vida: ${this.vida} | 💪 Fuerza: ${this.fuerza}`; } } // Creando personajes let guerrero = new Personaje("Thorin", 120, 25); let mago = new Personaje("Elara", 80, 18); console.log(guerrero.estado()); console.log(guerrero.atacar(mago));
