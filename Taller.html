🧠 Clases y Objetos

Una clase es una plantilla. Un objeto es una instancia con atributos y métodos.

// Clase en pseudocódigo / JS class Estudiante { constructor(nombre, edad) { this.nombre = nombre; this.edad = edad; } saludar() { return "Hola, soy " + this.nombre; } }

⚙️ Atributos y Métodos

Atributos: características (nombre, nota). Métodos: acciones (calcularPromedio, mostrarInfo).

🎯 Principios: Abstracción, Encapsulamiento, Herencia, Polimorfismo.

🤝 Aprendizaje Cooperativo: Método “Roles POO”

Cada equipo de 4 personas tendrá roles específicos. Deberán diseñar una clase "ProductoDigital" que incluya atributos y métodos.

👩‍💻 Programador/a

Escribe la sintaxis de la clase, atributos y métodos. Asegura que compile en pseudocódigo o JavaScript.

📋 Diseñador/a de Atributos

Define qué datos necesita la clase (nombre, precio, stock, categoría) y sus tipos (string, number, boolean).

🧪 Tester / Depurador

Crea casos de prueba: instancias de objetos y verifica métodos como aplicarDescuento(), mostrarDetalles().

🎤 Presentador/a

Expone la solución al resto, explica las decisiones de diseño y muestra el código funcionando.

🎯 Tarea cooperativa (15 min): Construyan la clase ProductoDigital con:

  • Atributos: nombre, precio, descuento (0-100), stock
  • Método: calcularPrecioFinal() que devuelva precio - descuento%
  • Método: vender(cantidad) que reduzca stock
// Clase ProductoDigital (JavaScript orientado a objetos) class ProductoDigital { constructor(nombre, precio, descuento, stock) { this.nombre = nombre; this.precio = precio; this.descuento = descuento; this.stock = stock; } calcularPrecioFinal() { return this.precio - (this.precio * this.descuento / 100); } vender(cantidad) { if (cantidad <= this.stock) { this.stock -= cantidad; return `Venta exitosa. Stock restante: ${this.stock}`; } return "Stock insuficiente"; } } const libroPOO = new ProductoDigital("POO desde cero", 25, 15, 10); console.log(libroPOO.calcularPrecioFinal()); // 21.25

📌 Entregable final: Pseudocódigo / JS de la clase + al menos 3 objetos creados. Debe incluir encapsulamiento (getters/setters) y método toString.

// Clase base sugerida: Estudiante class Estudiante { #nombre; #notas; constructor(nombre, notas = []) { this.#nombre = nombre; this.#notas = notas; } agregarNota(nota) { this.#notas.push(nota); } getPromedio() { if (this.#notas.length === 0) return 0; return this.#notas.reduce((a,b)=>a+b,0)/this.#notas.length; } toString() { return `${this.#nombre} - Promedio: ${this.getPromedio().toFixed(2)}`; } } // Herencia: EstudianteInternacional
